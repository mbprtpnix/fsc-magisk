# Fira Sans Condensed for Magisk
Install **Fira Sans Condensed** as your default font systemless-ly  
![screenshot](screenshot.jpg)  
## Instructions
* Clone this repo
* ZIP it
* Install the ZIP file via Magisk or a custom recovery (e.g. TWRP)
* Reboot
